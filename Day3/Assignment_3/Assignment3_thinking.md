# Assignment3 の考え方

## 課題内容
括弧に対応しよう
例:  (3.0 + 4 * (2 − 1)) / 5

## 考え方
括弧単位で再帰構造を作る。
- S -> (S)S | ε

### 仕様
- 括弧の中は1つの式として評価され、その評価結果は1つの数値として扱われるものとする
    - 数字 + 演算子 + 数字（例: (3 + 4)）
    - 数字 + 演算子 + 括弧（例: (3 + (4 * 2))）
    - 括弧 + 演算子 + 数字（例: ((1 + 2) * 3)）
    - 括弧 + 演算子 + 括弧（例: ((1 + 2) * (3 - 4))）

### 実装したテストコード
- S -> ((S))
    - test("(1+2)")
    - test("(1.0+(2.1-3))")
    - test("((8*2)+1)")
    - 和が優先
        - test("((1+2)*3)")
    - 差が優先
        - test("(4/(-3+1))")
- S -> (S)(S)
    test("(8*3)+(1/1)")
    test("(1+3)*(3-7)")
    test("(4/2)-(2+1)")
- S -> (S)S
    test("10*(1+1)")
    test("(7-3)/2")
- 複雑な計算
    test("(1-2)+3*4*(5-6)-(7+8)/5")
